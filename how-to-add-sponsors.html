üéØ Quick Overview

Adding sponsors is as simple as:

  1. Save the sponsor's logo image
  2. Edit the sponsors.json file
  3. Refresh the website - Done! üéâ

üìã Step-by-Step Instructions

1 Get the Sponsor Logo

Ask the sponsor for their logo. Best formats are:

  • PNG (recommended - supports transparent backgrounds)
  • JPG or JPEG
  • SVG (vector format)
2 Save Logo to the Right Folder

Save the logo file to: images/sponsors/

Use a simple filename like: company-name.png

Example: images/sponsors/vienna-dental.png

3 Open sponsors.json

Find and open the file called sponsors.json

You can use:

  • TextEdit (Mac)
  • Notepad (Windows)
  • Any text editor
4 Add the Sponsor Information

Copy this template and fill in the details:

{
  "name": "Company Name Here",
  "tier": "gold",
  "logo": "images/sponsors/filename.png",
  "description": "Short description",
  "website": "https://website.com"
}

Common Mistakes to Avoid:
  • ‚ùå Missing commas between entries
  • ‚ùå Extra comma after the last entry
  • ‚ùå Misspelling the tier name (must be exactly: platinum, gold, silver, or bronze)
  • ‚ùå Wrong file path for logo (make sure it matches where you saved it)

üîç Testing Your Changes

After editing sponsors.json:

  1. Save the file
  2. Open sponsorship.html in your web browser
  3. Refresh the page (press F5 or Cmd+R)
  4. Your new sponsor should appear!

üÜò Troubleshooting

Sponsor isn't showing up?

  • Check that the logo file path is correct
  • Make sure the tier is spelled correctly (all lowercase)
  • Verify you saved the sponsors.json file
  • Try refreshing the browser (Ctrl+F5 or Cmd+Shift+R for hard refresh)

Sponsor shows but no logo?

  • Check that the logo file is in images/sponsors/
  • Verify the filename matches exactly (including file extension)
  • Make sure the logo path in JSON is correct

Website looks broken?

  • Check for syntax errors in sponsors.json
  • Verify you have commas between entries
  • Use a JSON validator: jsonlint.com
